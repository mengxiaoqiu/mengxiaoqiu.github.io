Markdown学习笔记

语法

标题

在想要设置为标题的文字前面加#来表示，标准语法一般在#后跟个空格再写文字
一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

字体

  1. 加粗： 两个星号加粗
  2. 斜体： 一个星号斜体
  3. 斜体加粗：三个星号斜体和加粗
  4. 删除线： 两个删除线~~

分割线

三个或者三个以上的 - 或者 * 都可以

代码

1.单行代码：代码之间分别用一个反引号`包起来

printf("hello world")

2.代码块：三个反引号`开启代码块，反引号后可指定具体语言

列表

1.无序列表：使用星号、加号或是减号 加上空格作为列表标记

  • red
  • green
  • blue

2.有序列表：使用数字接着一个英文句点+空格

  1. 第一点
  2. 第二点
  3. 第三点

颜色

  1. 使用背景色：两个等号括起来

    ==使用默认背景色==

  2. 修改背景色：

    背景色的设置是按照十六进制颜色值：#C0FF3E

    ==修改背景色==
